# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it by [opening a GitHub issue](https://github.com/amsavarthan/android-tools-mcp/issues).

## Scope

This plugin runs locally inside Android Studio and communicates only over `127.0.0.1`. It does not make external network requests, store credentials, or handle authentication. The MCP server is accessible only from the local machine.
